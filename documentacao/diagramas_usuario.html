Diagramas de Usuário - Sistema de Controle de Serviços

Esta página apresenta diagramas visuais que utilizam formas geométricas para representar os usuários do sistema e suas funcionalidades. Os diagramas mostram de forma clara e intuitiva o que cada tipo de usuário pode fazer no sistema, incluindo cadastro, definição de níveis de acesso, e outras operações.

  • Funcionalidades do Usuário
  • Perfis de Usuários
  • Fluxo do Sistema

Diagrama de Funcionalidades do Usuário

Este diagrama utiliza formas geométricas para representar as principais funcionalidades disponíveis para os usuários do sistema. O círculo central representa o usuário, enquanto as demais formas mostram as diferentes ações que podem ser realizadas.

Legenda:

  • Círculo Azul: Representa o usuário (entidade principal)
  • Retângulos Azul-claro: Funcionalidades de cadastro
  • Hexágonos Verdes: Níveis de acesso disponíveis
  • Triângulos Amarelos: Permissões no sistema
  • Diamantes Vermelhos: Processos que podem ser executados

Diagrama de Perfis de Usuários

Este diagrama mostra os diferentes perfis de usuários do sistema e as funcionalidades específicas disponíveis para cada um. Cada círculo representa um tipo de usuário, e os retângulos conectados mostram as ações que cada perfil pode realizar.

Perfis de Usuários:

  • Administrador: Acesso total ao sistema, incluindo gerenciamento de usuários e configurações
  • Técnico: Responsável pelo atendimento e execução das ordens de serviço
  • Atendente: Realiza cadastro de clientes e criação de ordens de serviço
  • Cliente: Acesso externo para consulta e aprovação de serviços

Diagrama de Fluxo do Sistema

Este diagrama ilustra o fluxo de trabalho completo do sistema, desde o login até a geração de relatórios. As diferentes formas geométricas representam as etapas do processo e como os diferentes perfis de usuários interagem com o sistema.

Etapas do Fluxo:

  • Início: Acesso ao sistema
  • Login: Autenticação do usuário
  • Nível de Acesso: Determinação do perfil e permissões
  • Ações Específicas: Funcionalidades disponíveis para cada perfil
  • Convergência: Integração das diferentes ações
  • Relatórios: Geração de informações consolidadas
  • Fim: Conclusão do processo
