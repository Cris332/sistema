Manual do Usuário

Sobre o Sistema

Bem-vindo ao Sistema de Controle de Serviços! Esta ferramenta foi desenvolvida para facilitar o gerenciamento completo de ordens de serviço, controle de entregas e geração de relatórios para sua empresa.

Este sistema oferece:

  • Gerenciamento de Ordens de Serviço - Cadastre, edite, pesquise e imprima ordens de serviço
  • Relatórios Detalhados - Gere análises completas de faturamento, produtividade e desempenho
  • Controle de Entregas - Acompanhe entregas e retiradas de equipamentos com rastreamento em tempo real
  • Opções de Personalização - Configure o sistema de acordo com suas preferências e necessidades

Nota: Este manual foi elaborado para guiar usuários de todos os níveis de experiência. Navegue pelas abas acima para encontrar informações detalhadas sobre cada funcionalidade.

Navegação no Sistema

O sistema está dividido em cinco seções principais, acessíveis pela barra de navegação superior:

  • Ordem de Serviço - Página inicial para gerenciamento das ordens de serviço
  • Relatórios - Geração e visualização de relatórios e gráficos
  • Entregas - Controle de entregas e retiradas de equipamentos
  • Ajuda - Este manual do usuário
  • Opções - Configurações e personalização do sistema

Requisitos do Sistema

Para utilizar este sistema de forma adequada:

  • Navegador web atualizado (Chrome, Firefox, Edge ou Safari)
  • Resolução mínima de tela recomendada: 1024x768
  • JavaScript habilitado
  • Para mapas e rastreamento de entregas: acesso à Internet

Dica: Para melhor experiência, você pode personalizar a aparência do sistema na seção "Opções", incluindo temas escuros, alto contraste e tamanho de fonte.

Gerenciamento de Ordens de Serviço

O módulo de Ordem de Serviço (OS) é o núcleo do sistema, permitindo o registro e controle completo dos serviços técnicos realizados. A página inicial do sistema apresenta o formulário de Ordem de Serviço com todos os campos necessários.

Criando uma Nova Ordem de Serviço

Para cadastrar uma nova Ordem de Serviço:

  1. Na página inicial, clique no botão (Novo) na barra de ferramentas
  2. Os campos serão limpos para você inserir uma nova OS
  3. Preencha os campos obrigatórios (marcados com *):
    • Número OS - Identificador único da ordem de serviço
    • Data - Data de abertura da OS
    • Equipamento - Tipo/modelo do equipamento
    • Defeito - Descrição do problema relatado
    • Id Cliente - Código de identificação do cliente
  4. Preencha os campos opcionais conforme necessário:
    • Serviço - Descrição do serviço realizado ou a realizar
    • Técnico - Nome do técnico responsável
    • Valor Total - Valor cobrado pelo serviço
    • Cliente - Nome do cliente (preenchido automaticamente ao informar o Id)
  5. Clique no botão (Salvar) para registrar a OS

Dica: O sistema pode gerar automaticamente um número sequencial para cada nova OS, mas você pode alterá-lo se necessário.

Pesquisando Ordens de Serviço

Para buscar ordens de serviço já cadastradas no sistema:

  1. Na página inicial, clique no botão (Pesquisar) na barra de ferramentas
  2. Uma janela de pesquisa será aberta com vários campos de filtro:
    • Número OS - Para localizar uma OS específica pelo número
    • Data Inicial/Final - Para filtrar OS por período
    • Cliente - Para buscar OS de um cliente específico
    • Status - Para filtrar por status (em andamento, concluída, etc.)
    • Técnico - Para listar OS atribuídas a um técnico específico
    • Equipamento - Para buscar por tipo de equipamento
  3. Preencha um ou mais campos para filtrar sua busca
  4. Clique em "Pesquisar" para executar a busca
  5. Os resultados aparecerão em uma tabela abaixo
  6. Clique em uma OS na tabela de resultados para abri-la no formulário principal

Dica: Se você não preencher nenhum campo de filtro e clicar em "Pesquisar", o sistema mostrará todas as ordens de serviço mais recentes.

Editando uma Ordem de Serviço

Para modificar os dados de uma ordem de serviço existente:

  1. Primeiro, localize a OS que deseja editar usando a função de pesquisa
  2. Com a OS carregada no formulário principal, modifique os campos necessários:
    • Atualize o "Defeito" ou "Serviço" com informações mais detalhadas
    • Modifique o "Técnico" responsável, se necessário
    • Ajuste o "Valor Total" conforme o serviço realizado
  3. Após fazer as alterações, clique no botão (Salvar) para atualizar os dados
  4. O sistema confirmará que as alterações foram salvas com sucesso

Atenção: Tenha cuidado ao editar ordens de serviço. Modificar informações importantes como o número da OS pode causar inconsistências no sistema.

Excluindo uma Ordem de Serviço

Para remover uma ordem de serviço do sistema:

  1. Primeiro, localize a OS que deseja excluir usando a função de pesquisa
  2. Com a OS carregada no formulário principal, clique no botão (Excluir)
  3. Uma mensagem de confirmação será exibida
  4. Confirme a exclusão para remover permanentemente a OS do sistema

Atenção: Esta ação não pode ser desfeita! Quando você exclui uma ordem de serviço, todos os dados relacionados a ela são removidos permanentemente do sistema.

Imprimindo uma Ordem de Serviço

Para imprimir uma ordem de serviço:

  1. Localize a OS que deseja imprimir usando a função de pesquisa
  2. Com a OS carregada no formulário principal, clique no botão (Imprimir)
  3. O sistema preparará uma versão formatada para impressão da OS
  4. A janela de impressão do navegador será aberta
  5. Selecione a impressora desejada e ajuste as configurações de impressão
  6. Clique em "Imprimir" para enviar o documento para a impressora

Dica: Na seção "Opções" do sistema, você pode personalizar o formato de impressão das OS, incluindo ou removendo o logotipo da empresa, data e hora, cabeçalho e rodapé.

Cadastro de Clientes

Para cadastrar um novo cliente no sistema:

  1. No menu principal, clique em "Clientes" e depois em "Novo Cliente".
  2. Preencha os dados do formulário:
    • Nome/Razão Social
    • CPF/CNPJ
    • Telefone
    • E-mail
    • Endereço completo
    • Tipo de cliente (Pessoa Física ou Jurídica)
  3. Clique em "Salvar" para finalizar o cadastro.

Nota: Os campos marcados com asterisco (*) são obrigatórios e devem ser preenchidos para concluir o cadastro.

Sistema de Relatórios

O módulo de Relatórios oferece ferramentas para análise de desempenho, controle financeiro e acompanhamento das atividades. Com ele, você pode gerar diversos tipos de relatórios para obter insights valiosos sobre o funcionamento do seu negócio.

Tipos de Relatórios Disponíveis

O sistema oferece os seguintes tipos de relatórios para análise de dados:

  • Relatório de Ordens de Serviço - Lista detalhada de todas as OS registradas no período selecionado, incluindo informações sobre cliente, equipamento, valor e status
  • Relatório de Faturamento - Análise financeira completa mostrando o faturamento por período, cliente, tipo de serviço ou técnico responsável
  • Relatório de Status - Visão geral da distribuição de OS por status (em andamento, concluída, cancelada, etc.)
  • Relatório de Produtividade - Análise da quantidade de OS concluídas por período ou por técnico, permitindo avaliar a eficiência da equipe

Dica: Combine diferentes filtros para obter análises mais específicas, como faturamento por técnico em um determinado período.

Gerando um Relatório

Para gerar um relatório no sistema:

  1. Acesse a seção "Relatórios" no menu principal
  2. Selecione o tipo de relatório desejado no campo "Tipo de Relatório"
  3. Especifique o período de análise nos campos "Data Inicial" e "Data Final"
  4. Se disponível, selecione filtros adicionais relevantes para o tipo de relatório escolhido:
    • Cliente
    • Técnico
    • Status
    • Tipo de Serviço
  5. Clique no botão "Gerar Relatório" para processar os dados
  6. O relatório será exibido na tela com tabelas e gráficos

Nota: Relatórios com muitos dados podem levar mais tempo para serem gerados. Para melhor performance, tente limitar o período de análise ou adicionar filtros mais específicos.

Visualizando Gráficos

Os relatórios incluem representações gráficas dos dados para facilitar a análise. Esses gráficos são gerados automaticamente de acordo com o tipo de relatório:

  • Gráficos de Barras - Utilizados para comparar valores entre categorias diferentes, como faturamento por técnico ou quantidade de OS por status
  • Gráficos de Linha - Mostram a evolução de valores ao longo do tempo, como faturamento mensal ou número de OS por semana
  • Gráficos de Pizza - Representam a proporção entre diferentes categorias, como distribuição de OS por tipo de serviço

Para interagir com os gráficos:

  • Passe o mouse sobre os elementos do gráfico para ver informações detalhadas
  • Clique nas legendas para ocultar/mostrar categorias específicas
  • Use os controles de zoom, quando disponíveis, para focar em áreas específicas
Exportando Relatórios

Após gerar um relatório, você pode exportá-lo em diferentes formatos para uso externo:

  1. Gere o relatório desejado com os filtros apropriados
  2. Na parte inferior da página de relatório, encontre os botões de exportação:
    • Exportar para PDF - Cria um documento PDF com tabelas e gráficos, ideal para impressão ou compartilhamento
    • Exportar para Excel - Gera uma planilha Excel contendo os dados do relatório, permitindo análises adicionais
  3. Clique no formato desejado e aguarde o download do arquivo
  4. O arquivo será salvo automaticamente na pasta de downloads do seu navegador

Dica: O formato PDF é ideal para apresentações e relatórios formais, enquanto o formato Excel é melhor para análises adicionais e manipulação dos dados.

Gerenciamento de Entregas

O módulo de Entregas do sistema permite o controle completo da logística de entrega e retirada de equipamentos, oferecendo rastreamento em tempo real e registro de todas as operações.

Tipos de Operações

O sistema oferece dois tipos de operações logísticas:

  • Entrega - Utilizada quando um equipamento reparado será devolvido ao cliente
  • Retirada - Utilizada quando um equipamento será coletado na localização do cliente para manutenção

Selecione o tipo apropriado de operação usando os botões de rádio no início do formulário de Entregas.

Registrando uma Nova Operação

Para registrar uma nova entrega ou retirada:

  1. Acesse a seção "Entregas" no menu principal
  2. Selecione o tipo de operação (Entrega ou Retirada)
  3. Preencha os campos do formulário:
    • Endereço - Endereço completo onde o equipamento será entregue ou retirado
    • Cliente - Nome do cliente
    • Item - Descrição do equipamento ou ferramenta
    • Observações - Informações adicionais relevantes para a operação
  4. Clique no botão "Iniciar Entrega" ou "Iniciar Retirada" para começar a operação

Nota: O sistema usa o endereço informado para calcular a rota no mapa e o tempo estimado de entrega ou retirada.

Utilizando o Mapa

O módulo de Entregas inclui um mapa interativo que exibe:

  • A localização de origem (geralmente sua empresa)
  • A localização de destino (endereço do cliente)
  • A rota calculada entre os dois pontos
  • O tempo estimado de chegada

Para interagir com o mapa:

  • Use os controles de zoom (+/-) para aproximar ou afastar a visualização
  • Arraste o mapa para navegar por diferentes áreas
  • Clique nos marcadores para ver informações detalhadas sobre cada ponto
Histórico e Status

O sistema mantém registro completo de todas as operações de entrega e retirada, permitindo:

  • Visualizar o status atual de cada operação
  • Acompanhar o histórico completo de ações
  • Registrar momentos importantes como início, chegada e conclusão

Os status possíveis para uma operação incluem:

  • Aguardando início - Operação registrada mas ainda não iniciada
  • Em andamento - Operação iniciada e em progresso
  • Concluída - Operação finalizada com sucesso
  • Cancelada - Operação cancelada antes da conclusão

Dica: Você pode cancelar uma operação em andamento utilizando o botão "Cancelar", que será ativado após o início da entrega ou retirada.

Opções do Sistema

A página de Opções permite personalizar diversos aspectos do sistema, adaptando-o às suas necessidades específicas. As configurações estão organizadas em categorias para fácil navegação.

Aparência

As opções de aparência permitem personalizar a interface visual do sistema:

  • Tema Escuro - Alterna entre o tema claro (padrão) e o tema escuro, ideal para uso noturno ou ambientes com pouca luz
  • Fonte Grande - Aumenta o tamanho do texto em todo o sistema para melhor legibilidade
  • Alto Contraste - Ativa um modo de alto contraste para usuários com dificuldades visuais
  • Tamanho da Fonte - Controle deslizante que permite ajustar o tamanho exato da fonte de 12px a 24px

Dica: O tema escuro é especialmente útil para reduzir o cansaço visual durante uso prolongado do sistema.

Acessibilidade

As opções de acessibilidade tornam o sistema mais inclusivo:

  • Modo de Fala - Ativa a leitura automática dos elementos da interface quando você passa o mouse sobre eles. O sistema utiliza uma voz feminina em português brasileiro (quando disponível) para descrever:
    • Textos e cabeçalhos das páginas
    • Botões e suas funções específicas
    • Links e para onde eles direcionam
    • Campos de formulário e seu propósito
    • Imagens e seus conteúdos
  • Velocidade da Fala - Controle deslizante que permite ajustar a velocidade de leitura do texto, de lenta (0.5x) a rápida (2.0x)
  • Leitor de Tela - Otimiza a interface para funcionar melhor com leitores de tela para deficientes visuais
  • Navegação por Teclado - Ativa recursos de navegação avançada por teclado, incluindo atalhos Alt e indicação visual de foco
  • Reduzir Animações - Minimiza ou elimina todas as animações da interface para usuários sensibilizados por movimentos

Dica: O modo de fala é especialmente útil para pessoas com deficiência visual parcial ou dificuldade de leitura, permitindo ouvir o conteúdo da tela sem necessidade de um leitor de tela completo.

Navegação por Teclado e Atalhos

O sistema oferece atalhos de teclado avançados para navegação rápida e eficiente sem necessidade do mouse. Para ativar esses recursos, marque a opção "Navegação por Teclado" na página de Opções.

Todos os recursos de navegação por teclado foram projetados para funcionar com leitores de tela, melhorando significativamente a acessibilidade do sistema.

Navegação entre Páginas (Alt + Número)

Pressione a tecla Alt junto com o número correspondente para navegar rapidamente entre as páginas principais do sistema:

Atalho Destino Descrição
Alt + 1 Página inicial / Ordens de Serviço Acessa a página principal de gerenciamento de ordens de serviço
Alt + 2 Clientes Acessa o cadastro e gerenciamento de clientes
Alt + 3 Relatórios Acessa o sistema de relatórios e análises
Alt + 4 Opções Acessa a página de configurações do sistema
Alt + 5 Ajuda Acessa esta página de ajuda
Alt + 0 Sistema de entregas Acessa o módulo de gerenciamento de entregas

Ações Comuns (Alt + Letra)

Use esses atalhos para executar ações frequentes em qualquer página do sistema:

Atalho Ação Disponível em
Alt+S Salvar Qualquer formulário ou página de configurações
Alt+P Pesquisar Páginas com opção de pesquisa
Alt+N Novo Páginas de cadastro (OS, clientes, etc.)
Alt+X Excluir Páginas de cadastro com opção de exclusão
Alt+B Backup Página de opções
Alt+L Limpar Qualquer formulário ou tela de pesquisa
Alt+I Imprimir Páginas com opção de impressão

Navegação com Teclado

Quando a navegação por teclado está ativada, você pode usar as teclas de seta para navegar entre elementos da interface:

  • Tab - Navega sequencialmente entre todos os elementos interativos
  • Shift+Tab - Navega na ordem inversa
  • Setas ← → - Navega entre itens de menu horizontal ou ajusta valores em controles deslizantes
  • Setas ↑ ↓ - Navega entre itens de menu vertical e opções em listas
  • Enter - Ativa o elemento selecionado (botões, links, etc.)
  • Espaço - Alterna opções (checkboxes, radio buttons)
  • Esc - Fecha menus abertos, caixas de diálogo ou cancela a operação atual
  • Home/End - Vai para o início ou final de uma lista ou página

Teclas de Acesso Rápido para Elementos Principais

Para acessar elementos principais da interface rapidamente, use a combinação Alt+Shift+Letra:

  • Alt+Shift+C - Foca no conteúdo principal da página
  • Alt+Shift+M - Foca no menu de navegação principal
  • Alt+Shift+F - Foca no primeiro campo de formulário
  • Alt+Shift+B - Foca no botão principal da página (geralmente Salvar ou Enviar)

Modo de Fala e Leitura de Tela

O sistema oferece um modo de fala que lê o texto dos elementos quando o mouse passa sobre eles ou quando são focados pelo teclado:

  • Ative o modo de fala na página de Opções, na seção "Acessibilidade"
  • Ajuste a velocidade da fala usando o controle deslizante
  • A leitura é automática ao navegar com o mouse ou teclado quando o modo está ativado
  • O sistema seleciona automaticamente uma voz feminina em português brasileiro, quando disponível

Nota sobre compatibilidade: Os atalhos Alt podem variar de acordo com o navegador utilizado. Em alguns navegadores, pode ser necessário pressionar Alt+Shift+Letra.

Dica: Quando a navegação por teclado está ativa, um indicador visual destaca o elemento que está em foco, tornando mais fácil identificar onde você está na página.

Impressão

As opções de impressão controlam como os documentos serão formatados quando impressos:

  • Imprimir Logotipo - Inclui ou remove o logotipo da empresa nas impressões de OS e relatórios
  • Imprimir Data/Hora - Adiciona ou remove a data e hora de impressão nos documentos
  • Incluir Cabeçalho e Rodapé - Controla a exibição de informações da empresa no cabeçalho e rodapé das páginas impressas

Dica: Para documentos oficiais enviados a clientes, recomendamos manter todas as opções de impressão ativadas para uma apresentação profissional.

Notificações

Configure como e quando o sistema deve enviar alertas:

  • Notificar Nova OS - Exibe uma notificação quando uma nova ordem de serviço é registrada
  • Notificar OS Concluída - Alerta quando uma ordem de serviço é marcada como concluída
  • Notificar Novo Cliente - Avisa quando um novo cliente é cadastrado no sistema
  • Tempo de Notificação - Controle deslizante que define por quantos segundos as notificações ficam visíveis na tela
Backup

As opções de backup ajudam a proteger seus dados contra perdas:

  • Backup Automático - Ativa ou desativa a realização automática de backups periódicos
  • Backup na Nuvem - Define se os backups devem ser armazenados localmente ou sincronizados com um serviço de armazenamento na nuvem
  • Horário do Backup - Define em qual horário do dia o sistema deve realizar o backup automático
  • Backup Manual - Botão para realizar imediatamente um backup completo do sistema

Atenção: Recomendamos manter o backup automático sempre ativado e realizar backups manuais adicionais antes de fazer grandes alterações no sistema.

Salvando as Configurações

Depois de fazer suas personalizações, você precisa salvá-las:

  1. Ajuste todas as configurações desejadas nas várias categorias
  2. Clique no botão "Salvar Opções" na parte inferior da página
  3. Uma notificação confirmará que suas configurações foram salvas
  4. As novas configurações entrarão em vigor imediatamente

Se desejar restaurar todas as configurações para os valores padrão, clique no botão "Restaurar Padrão".
