Manual do Usuário

Introdução ao Sistema

Bem-vindo ao Sistema de Controle de Serviços! Esta ferramenta foi desenvolvida para facilitar o gerenciamento completo de ordens de serviço em sua empresa, permitindo o cadastro, consulta, edição e impressão de OS de forma simples, rápida e eficiente.

Com este sistema, você poderá:

  • Cadastrar, editar e excluir ordens de serviço
  • Cadastrar e gerenciar informações de clientes
  • Pesquisar ordens de serviço por diversos critérios
  • Gerar relatórios e análises de desempenho
  • Imprimir ordens de serviço para documentação física

Nota: Este manual foi elaborado para guiar usuários de todos os níveis de experiência. Se tiver dúvidas específicas, consulte a seção correspondente.

Requisitos do Sistema

Para utilizar este sistema, você precisará de:

  • Navegador web atualizado (Chrome, Firefox, Edge ou Safari)
  • Conexão com a internet
  • Resolução mínima de tela de 1024x768
  • JavaScript habilitado
Cadastro de Ordens de Serviço

Para cadastrar uma nova ordem de serviço, siga os passos abaixo:

  1. Na página inicial, clique no botão "Nova OS".
  2. Preencha os dados do cliente (você pode selecionar um cliente já cadastrado ou inserir um novo).
  3. Descreva o serviço a ser realizado no campo "Descrição do Serviço".
  4. Selecione o tipo de serviço na lista suspensa.
  5. Informe o status inicial da ordem de serviço.
  6. Adicione as peças/materiais utilizados, se houver.
  7. Informe o valor do serviço.
  8. Defina a data prevista para conclusão.
  9. Clique em "Salvar OS" para finalizar o cadastro.

Dica: Você pode utilizar a tecla TAB para navegar entre os campos do formulário de forma mais rápida.

Pesquisa de Ordens de Serviço

Para pesquisar ordens de serviço existentes no sistema:

  1. Acesse a tela de "Pesquisa de OS" através do menu principal.
  2. Utilize os filtros disponíveis para refinar sua busca:
    • Número da OS
    • Nome do cliente
    • CPF/CNPJ do cliente
    • Data de abertura
    • Status da OS
    • Tipo de serviço
  3. Clique em "Pesquisar" para visualizar os resultados.
  4. Os resultados serão exibidos em uma tabela com as principais informações de cada OS.
  5. Para visualizar mais detalhes de uma OS específica, clique no ícone de Detalhes na linha correspondente.

Nota: Você pode deixar os campos de filtro em branco para listar todas as ordens de serviço do sistema, mas isso pode tornar a busca mais lenta se houver muitos registros.

Edição de Ordens de Serviço

Para editar uma ordem de serviço existente:

  1. Localize a OS que deseja editar utilizando a função de Pesquisa.
  2. Clique no ícone de Edição (lápis) na linha correspondente à OS.
  3. O sistema abrirá o formulário de edição com os dados atuais da OS.
  4. Faça as alterações necessárias nos campos disponíveis.
  5. Atualize o status da OS se necessário.
  6. Adicione observações sobre as alterações no campo "Observações".
  7. Clique em "Salvar Alterações" para confirmar.

Atenção: Algumas informações da OS não podem ser alteradas após sua criação, como o número da OS e a data de abertura.

Exclusão de Ordens de Serviço

Para excluir uma ordem de serviço do sistema:

  1. Localize a OS que deseja excluir utilizando a função de Pesquisa.
  2. Clique no ícone de Exclusão (lixeira) na linha correspondente à OS.
  3. O sistema exibirá uma mensagem de confirmação.
  4. Clique em "Confirmar" para prosseguir com a exclusão ou "Cancelar" para desistir.

Atenção: A exclusão de uma OS é permanente e não pode ser desfeita. Certifique-se de que realmente deseja excluir o registro antes de confirmar.

Impressão de Ordens de Serviço

Para imprimir uma ordem de serviço:

  1. Localize a OS que deseja imprimir utilizando a função de Pesquisa.
  2. Clique no ícone de Impressão (impressora) na linha correspondente à OS.
  3. O sistema exibirá uma prévia da OS formatada para impressão.
  4. Verifique se todas as informações estão corretas.
  5. Clique em "Imprimir" para enviar para a impressora ou "Cancelar" para voltar.
  6. Na janela de impressão do navegador, selecione a impressora desejada e as configurações de impressão.
  7. Clique em "Imprimir" para finalizar.

Dica: Você também pode salvar a OS como um arquivo PDF utilizando a opção "Salvar como PDF" na janela de impressão do navegador.

Cadastro de Clientes

Para cadastrar um novo cliente no sistema:

  1. No menu principal, clique em "Clientes" e depois em "Novo Cliente".
  2. Preencha os dados do formulário:
    • Nome/Razão Social
    • CPF/CNPJ
    • Telefone
    • E-mail
    • Endereço completo
    • Tipo de cliente (Pessoa Física ou Jurídica)
  3. Clique em "Salvar" para finalizar o cadastro.

Nota: Os campos marcados com asterisco (*) são obrigatórios e devem ser preenchidos para concluir o cadastro.

Relatórios Disponíveis

O sistema oferece diversos relatórios para análise e acompanhamento dos serviços:

  • Relatório de OS por Período - Exibe todas as OS abertas em um determinado período
  • Relatório de Faturamento - Mostra o faturamento por período, cliente ou tipo de serviço
  • Relatório de Clientes - Lista todos os clientes cadastrados e suas estatísticas
  • Relatório de Produtividade - Analisa a quantidade de OS concluídas por período
